Project 1

Health App Prototype (Figma)

A health-themed login and multi-screen prototype designed for clarity, comfort, and easy navigation.

Figma UI Design Prototyping

Overview

This project was built as part of a multi-day Figma tutorial where the goal was to complete a functional, clickable prototype with a customized health theme. I focused on consistent typography, spacing, and a calm visual style that supports readability.

What I built

  • A customized health-related login screen
  • Two additional health-themed screens with original layout and content
  • A menu overlay screen to support navigation

Process

  • Planned a simple user flow (login → main screen → secondary screen → menu overlay)
  • Designed consistent UI components (buttons, headers, spacing)
  • Linked screens to create a smooth clickable prototype
